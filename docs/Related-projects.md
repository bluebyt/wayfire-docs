Here is a list of projects which are related to Wayfire.

See also the [wlroots wiki](https://gitlab.freedesktop.org/wlroots/wlroots/-/wikis/Projects-which-use-wlroots) and [awesome-wlroots]( https://github.com/solarkraft/awesome-wlroots) for a list of projects which utilize `wlr` protocols, which are also supported in Wayfire.

## Desktop environments

- [Paper](https://gitlab.com/cubocore/paper)
- [DesQ](https://desq.gitlab.io//)
- [Genesis](https://github.com/ExpidusOS/genesis)
- [RDE](https://gitlab.com/eocene/rde)
- [carbonOS](https://gitlab.com/carbonOS/gde) (project archived)
- [waysmoke](https://github.com/DankBSD/waysmoke) (project archived)

## Linux distributions

- ~~[Garuda Linux Wayfire](https://garudalinux.org/downloads.html)~~ - An Arch based Linux distro with a Wayfire based Desktop Environment (was removed in 2022 [but might come back](https://forum.garudalinux.org/t/why-no-more-wayfire/17284/2))
- [Peux OS Wayfire](https://peux-os.netlify.app/twms) - An Arch based Linux distro with a Wayfire based Desktop Environment

## Panels

- [Waybar](https://github.com/Alexays/Waybar) - Highly customizable Wayland bar for Wlroots based compositors, written in C++.
- [Wf-panel (Gtk3)](https://github.com/WayfireWM/wf-shell) - A GTK4-based panel for Wayfire, written in C++.
- [Panorama Panel](https://roundabout-host.com/roundabout/panorama-panel) - Configurable, modular and traditional desktop panel, designed for Wayfire and written in Python.

## Plugins

### Plugin collections

- [wayfire-plugins-extra](https://github.com/WayfireWM/wayfire-plugins-extra) - Repository of additional Wayfire plugins
- [filters](https://github.com/soreau/filters) - Apply shaders to windows or outputs via ipc scripts
- [ammen99-plugins](https://github.com/ammen99/wayfire-plugins) - ammen99's personal plugins

### Decorations

- [pixdecor](https://github.com/soreau/pixdecor) - A highly configurable decorator plugin featuring antialiased rounded corners with shadows and optional animated effects
- ~[wayfire-firedecor](https://github.com/AhoyISki/wayfire-firedecor)~ - Alternative window decoration plugin for much greater configuration
- [cosmodecor](https://github.com/CosmicFusion/cosmodecor) - An advanced window decoration plugin for the wayfire window manager, continuation of firedecor.

### Switchers

- [wWP Switcher](https://github.com/wb9688/wwp-switcher) - Classic switcher plugin for Wayfire
- [qwf-overview](https://codeberg.org/qtea/qwf-overview) - Displays an overview of all open windows, similar to the GNOME Activity Overview

### Gestures

- [wstroke](https://github.com/dkondor/wstroke) - Port of Easystroke mouse gestures
- ~[wf-globalgestures](https://codeberg.org/valpackett/wf-globalgestures)~ - Allow clients to handle touchpad gestures globally via a special protocol - **not maintained**
- ~[wf-touchpad-gesture-drag](https://codeberg.org/valpackett/wf-touchpad-gesture-drag)~ - Implements 3-finger drag for multi-touch touchpads - **not maintained**

### Configuration backends

- ~[wf-gsettings](https://codeberg.org/valpackett/wf-gsettings)~ - GSettings configuration backend - **not maintained**
- [hjson-config-backend](https://gitlab.com/wayfireplugins/wf-config-hjson) - HJson based config backend for wayfire.

### Other stuff

- [laser-search](https://github.com/soreau/laser-search) - Search and launch applications with a stylish, intuitive, standalone application
- [wf-info](https://github.com/soreau/wf-info) - A simple wayfire plugin and program to get information from wayfire.
- [wf-mod2key](https://codeberg.org/valpackett/wf-mod2key) - xcape style "press modifier on its own to press another key combo" plugin
- [wf-wallpaper](https://codeberg.org/valpackett/wf-wallpaper) - Advanced wallpaper plugin
- [wayfire-shadows](https://github.com/timgott/wayfire-shadows) - Add shadows around windows
- [wayfire-kbdd-plugin](https://github.com/AlexJakeGreen/wayfire-kbdd-plugin) - Per-window keyboard layout plugin
- [wf-scud](https://gitlab.com/magus/wf-scud) - PaperWM-inspired window management plugin
- [wf-dynspaces](https://codeberg.org/valpackett/wf-dynspaces) - Dynamic workspace management with fullscreen-apps-as-workspaces
- [wayfire-toplevel-management](https://gitlab.com/wayfireplugins/windowmanager) - A more advanced version of the foreign-toplevel-management protocol.
